# Minecraft Sign Generator

Create Minecraft Signs that give players food.

## Demo

https://sign.wondersmc.net


## Run Locally

Clone the Project

```bash
  git clone https://github.com/wondersmc/Sign-Generator
```

Go to the Project Directory

```bash
  cd Sign-Generator
```

Install Dependencies

```bash
  npm install
```

Start the Server

```bash
  npm run dev
```
